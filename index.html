1: About

1.1 Open-source VS. Chrome web store

First I'll explain a little bit about the Chrome web store.

To add an extension to the store is quite a long and tedious process.
After this process the package can not be changed or updated.
So the beta vesrion that's in the store: hnschat - beta v0.0.3 cannot be changed.
Even when chrome updates and changes extension parameters.

The open-source option on the other hand..

The open-source version is a maintained repo providing the latest version.
You will have to download and add the latest version manually tho.
We will notify you on twitter and hnschat if there is a new release.
To install the extension from the github repo, please follow the guide below:
How to ~ install the latest version


1.2 Options

Options currently provides the following settings:

- Resolution

Set the reolution of the extension.
With the latest update, chrome has changed the resolution of the extension. The height is now limited at 597px instead of 600px.

- Source

Switch between https://hnschat/ and https://hns.chat/.

*To open the options menu, richt-click on the extension icon in your browser. Then click on options.

2: How to ~ install the latest version

2.1 ~ Get the package

  • Download package-{version}.zip from the repo's /files directory.

    • Latest: v0.0.4 In store: v0.0.3
  • Extract the files.

2.2 ~ Enable Developer mode

  • In your browser click: Menu -> Extensions
  • In the extension manager, check if Developer mode is enabled. If not, enable it.

*the buttun Load unpacked should be visible.


2.3 ~ Load the extension

  • Click the button: Load unpacked
  • Select the folder where you have extracted the files.
The extension should be added to your browser.
Just pin the icon and hnschat will be 1 click away.
Enjoy! Have a beautiful time!


3: How to ~ install from the chrome store

3.1 ~ Add the extension directly from the store

  • Go to: chrome.google.com/webstore/hnschat-beta/
  • Click the Add to browser button.
The extension should be added to your browser.
Just pin the icon and hnschat will be 1 click away.
Enjoy! Have a beautiful time!
